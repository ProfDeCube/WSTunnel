# WSTunnel
WSTunnel is a tool designed to make playing web based Archipelago games easier but will likely have other uses.
It acts as a local proxy that receives and sends messages from the provided target to the localhost websocket server.
## Usage
### Simple
Download the exe from the latest release.
Run the exe and provide the target websocket server address
check the terminal for the localhost address to use in the Archipelago game
### Running from Local
Clone the repository
Run an install of requirements.txt
Run WSTunnel.py
### Building
Clone the repo
Install the python module pyinstaller
Build the project using pyinstaller
Run the exe in the dist folder
